# zeyadhq Marketplace

Complete set of Claude Code plugins by Zeyad Ramadan for job application and UX/UI design workflows.

## Plugins

1. **uxui-methods-library** - UX/UI design methodology with proven methods and templates
2. **job-application-assistant** - Complete job application toolkit with strategic frameworks

## Installation

Add this marketplace to Claude Code:
```
git@github.com:zeyadhq/zeyadhq-marketplace.git
```
